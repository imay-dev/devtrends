## Getting Started

### Prerequisites

- Node.js 18+
- npm 9+
- GitHub Personal Access Token

### Installation

```bash
# Clone repository
git clone https://github.com/imay-dev/devtrends.git

# Install dependencies
npm install

# Create environment file
cp env.example .env.local
```
